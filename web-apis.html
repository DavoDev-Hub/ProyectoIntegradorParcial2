TECNOLOGÍAS DE WEB APIs

Comprendiendo las interfaces de programación de aplicaciones web

¿QUÉ ES UNA API?

Una API (interfaz de programación de aplicaciones) es un conjunto de reglas definidas que permiten que diferentes aplicaciones se comuniquen entre sí. Actúa como una capa intermedia que procesa las transferencias de datos entre sistemas.

Según Amazon Web Services (2023), las APIs son mecanismos que permiten a dos componentes de software comunicarse entre sí mediante un conjunto de definiciones y protocolos.

GET /api/users HTTP/1.1

Host: api.example.com

Authorization: Bearer token123

Accept: application/json

EJEMPLOS DE APIs POPULARES:

  • Las API de Google: Búsqueda, Gmail, Traductor o Maps
  • PayPal para procesamiento de pagos
  • API para incrustar videos de YouTube
  • WhatsApp Business API
  • APIs de redes sociales como Instagram y Facebook
FUNCIONAMIENTO DE UNA API

Las APIs funcionan como intermediarios entre diferentes sistemas, permitiendo que se comuniquen y compartan datos de manera eficiente y segura.

REST APIs

Las APIs REST (Representational State Transfer) son un estilo de arquitectura para diseñar aplicaciones en red. Utilizan métodos HTTP estándar como GET, POST, PUT y DELETE.

SOAP APIs

SOAP (Simple Object Access Protocol) es un protocolo de mensajería que permite a los programas que se ejecutan en diferentes sistemas operativos comunicarse.

GraphQL

GraphQL es un lenguaje de consulta para APIs y un tiempo de ejecución para cumplir esas consultas con los datos existentes, ofreciendo mayor flexibilidad que REST.
